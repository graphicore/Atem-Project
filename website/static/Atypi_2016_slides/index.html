Roadmap short-term (this year!)

  • Project loading and saving (almost there)
  • MOM Developer Tool: drag and drop control points (almost there)
  • Object Model inspection widget
  • Finish and polish the Bauhaus Emblem Font tool (almost there)
  • CPS-UI (almost there)
  • MOM Developer Tool: feature parity with current Metapolator

Roadmap medium-term (next year?)

  • Atem-Shell/Atem-IDE as a base for projects to start.
  • OMA/MOM Plugins
  • MOM Editor (importing from UFO is not sufficient)
  • Maybe: Metacomponents

After that:

  • Redo/Undo/History
  • Metapolator rewrite to run in the Atem-Shell
  • More new tools
  • Advance Metapolator
  • Improve CPS Tooling …
